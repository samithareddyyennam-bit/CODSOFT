# Handwritten Text Generation

## Overview

Handwritten Text Generation is a Python-based project that converts user-entered text into an image. The generated text is rendered on an image canvas and saved as a PNG file.

## Features

* Accepts user text input
* Generates a text image automatically
* Saves output as PNG image
* Simple and easy-to-use interface
* Lightweight implementation using Python

## Technologies Used

* Python
* Pillow (PIL)

## How It Works

1. User enters text.
2. The program creates a blank image.
3. The entered text is written onto the image.
4. The image is saved as a PNG file.
5. The generated image can be viewed and shared.

## Installation

Install the required package:

pip install -r requirements.txt

## Usage

Run the program:

python handwritten_text_generation.py

Enter the text when prompted.
