# Customer Churn Prediction

## Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company's service or continue using it. The project helps businesses identify at-risk customers and take preventive measures to improve customer retention.

## Features

* Data preprocessing and cleaning
* Customer churn prediction using Machine Learning
* Random Forest Classifier implementation
* Model accuracy evaluation
* Customer churn analysis through data visualization
* Interactive prediction system

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Dataset

The project uses the Telco Customer Churn Dataset containing customer information such as:

* Gender
* Senior Citizen Status
* Partner and Dependents
* Tenure
* Internet Service
* Contract Type
* Monthly Charges
* Total Charges
* Churn Status

## Machine Learning Algorithm

Random Forest Classifier

## Project Workflow

1. Load the customer churn dataset
2. Perform data preprocessing
3. Handle missing values
4. Convert categorical data into numerical format
5. Split data into training and testing sets
6. Train the Random Forest model
7. Evaluate model accuracy
8. Predict customer churn

## Results

* Model Accuracy: Approximately 78.5%
* Successfully predicts customer churn behavior
* Provides customer retention insights

## How to Run

1. Install required libraries:

pip install -r requirements.txt

2. Run the project:

python customer_churn_prediction.py
