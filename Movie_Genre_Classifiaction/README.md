# Movie Genre Classification

This project predicts movie genres using Machine Learning and Natural Language Processing (NLP).

## Features
- Predicts movie genre from description
- Uses Logistic Regression model
- Shows prediction confidence
- Displays top 10 movie genres graph

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Accuracy
Around 53.37 accuracy on movie genre prediction.

## How to Run

Install libraries:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python movie_genre_classification.py
```
